# 组件总览

🚀在一分钟内完成您的中后台管理页面

# 一级组件

  • ElemInput: 输入框
  • ElemInputNumber: 计数器
  • ElemAutocomplete: 输入匹配
  • ElemSelect: 下拉选择器
  • ElemCascader: 级联选择器
  • ElemDatePicker: 日期选择器
  • ElemListGroup: 备选项选择器

# 二级组件

  • ElemForm: 表单(包含一级组件)
  • ElemTable: 表格(包含一级组件)
  • ElemExtendTable: 分页表格

# 三级组件

  • ElemTablePage: 组合表单和表格并封装请求
